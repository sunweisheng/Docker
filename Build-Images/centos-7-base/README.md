# CentOS 7 Base

构建命令：

```shell
docker build -t bluersw/centos-7-base:1.0 .
```

运行命令：

```shell
docker run -it bluersw/centos-7-base:1.0 /bin/bash
```

Pull：

```shell
docker pull bluersw/centos-7-base:1.0
```
